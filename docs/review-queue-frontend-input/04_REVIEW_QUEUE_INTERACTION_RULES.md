# 待我处理页面｜交互规则

## 1. 事项类型 Tab

### Tab 列表

```text
全部 / 待确认 / 待审核 / 异常 / 继续执行
```

### 行为

点击 Tab 后：

```text
1. 更新当前类型筛选
2. 更新表格数据
3. 同步 URL query
4. 保留其他筛选条件，除非用户点击重置筛选
```

示例：

```ts
全部       → /tasks/reviews?assignee=me
待确认     → /tasks/reviews?assignee=me&type=CONFIRM
待审核     → /tasks/reviews?assignee=me&type=APPROVAL
异常       → /tasks/reviews?assignee=me&type=EXCEPTION
继续执行   → /tasks/reviews?assignee=me&type=RESUME
```

## 2. 搜索与筛选

### 搜索框

输入关键词后筛选以下字段：

```text
事项名称
来源任务名称
所属阶段名称
```

建议交互：

```text
输入后 debounce 300ms
Enter 立即搜索
清空输入后恢复列表
```

### 筛选项

```text
状态
类型
来源任务
发起人
优先级
截止时间
```

所有筛选应支持组合。

## 3. 排序

默认排序：

```text
优先级降序 + 截止时间升序 + 发起时间倒序
```

排序选项：

```text
优先级
截止时间
发起时间
来源任务
```

## 4. 批量选择

### 选择规则

```text
点击行首 checkbox 选择单行
点击表头 checkbox 全选当前页
跨页不默认保留选择，除非后续实现“跨页全选”
```

### 选择后显示批量操作条

```text
已选择 N 个事项
批量确认
批量转派
批量忽略
关闭
```

### 批量操作可用性

| 操作 | 可用于 |
|---|---|
| 批量确认 | 待确认事项 |
| 批量转派 | 所有未完成事项 |
| 批量忽略 | 待确认、低风险异常、继续执行提醒 |

如果选择的事项类型不支持某操作，按钮置灰并显示 tooltip。

## 5. 行级操作

| 类型 | 主按钮 | 行为 |
|---|---|---|
| 待确认 | 进入确认 | 进入确认详情页 |
| 待审核 | 去审核 | 进入审核详情页 |
| 异常 | 查看异常 | 进入异常处理页 |
| 继续执行 | 继续执行 | 进入任务执行页当前断点 |

## 6. 点击区域行为

```text
点击事项名称 → 进入事项详情页或打开右侧详情抽屉
点击来源任务 → 进入任务详情页
点击所属阶段 → 进入任务详情页并定位到该阶段
点击优先级 → 快速筛选该优先级
点击状态 → 快速筛选该状态
```

## 7. 右侧快捷视图

| 快捷视图 | 行为 |
|---|---|
| 高优先级事项 | priority=HIGH |
| 今日到期 | deadline=today |
| 异常事项 | type=EXCEPTION |
| 我发起的事项 | initiator=me |

## 8. 分页

分页需要与筛选、搜索联动。

```text
筛选变化后 page 重置为 1
pageSize 变化后 page 重置为 1
URL 同步 page / pageSize
```

## 9. 空状态

### 无待处理事项

```text
暂无待处理事项
所有任务都在正常推进。
```

按钮：

```text
返回任务中心
查看全部任务
```

### 筛选无结果

```text
没有符合条件的事项
请调整筛选条件后重试。
```

按钮：

```text
重置筛选
```

## 10. 加载态

使用骨架屏，覆盖：

```text
指标卡
筛选区
事项表格
右侧概览
```

不要只显示转圈。

## 11. 错误态

```text
事项数据加载失败，请稍后重试。
```

按钮：

```text
重新加载
```
