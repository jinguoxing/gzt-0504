# 找数问数执行页 v3 页面规格 v1.0

## 1. 页面定位

找数问数执行页是用户从 AI 工作台首页输入问题后，由 Xino 判断为 `data_qa` 意图后进入的页面。

页面职责：

```text
承接用户问题
直接回答问题
展示口径与证据
展示趋势/对比/拆解
支持继续追问
支持查看来源和 SQL
```

页面不是首页，不是聊天工具，不是治理类任务页，不是 BI 大屏。

---

## 2. 强制约束

```text
1. 页面顶部不得出现“当前问题输入区”或“当前问题展示卡”。
2. 用户问题只能作为问答流第一条用户消息出现。
3. 禁止传统聊天气泡、气泡尖角、胶囊泡泡。
4. 用户消息是右对齐轻量提问条。
5. Xino 消息是左对齐的普通文本说明 + 独立结果卡片组。
6. 首屏最多展示 4 个主要结果模块。
7. 明细表不得首屏大面积展开。
8. 右侧栏不得是任务计划，必须是答案依据。
```

---

## 3. 总体布局

## 6. 右侧答案依据栏

右侧栏标题：

```text
答案依据
```

### 当前口径：默认展开

```text
指标：采购金额
时间范围：2026-04-01 至 2026-04-30
统计范围：供应链业务域
计算口径：已审核采购订单含税金额
筛选条件：订单状态 = 已审核；订单类型 ≠ 作废
```

### 数据依据：默认展开

```text
数据源：supply_chain_prod
来源表：dwd_scm_purchase_order_line
来源字段：amount_tax_included
时间字段：po_approved_date
数据更新时间：2026-05-04 09:30
权限状态：可查询
数据质量：通过最近一次质量检查
可信度：高
```

### 查询计划 / SQL：默认折叠

折叠摘要：

```text
使用时间过滤 + 指标聚合，扫描 1 张事实表
```

展开后展示：

```sql
SELECT SUM(amount_tax_included) AS purchase_amount
FROM dwd_scm_purchase_order_line
WHERE po_approved_date BETWEEN '2026-04-01' AND '2026-04-30'
  AND order_status = 'approved';
```

### 追问上下文：默认折叠

```text
1. 上个月采购金额是多少？
2. 为什么本月采购金额上涨？
当前分析方向：采购金额变化归因
```
