# 06｜Page Spec｜字段确认 / 冲突处理 / 审核详情页

## 1. 页面目标

Review Detail 是人工介入页面，用于处理：

- 字段语义确认
- 冲突字段处理
- 异常字段审核
- 业务对象审核
- 标准映射审核

本版本以“字段确认与冲突处理”为主。

## 11. 交互规则

| 操作 | 结果 |
|---|---|
| 确认 A | 写入最终语义为 A，状态转已处理 |
| 确认 B | 写入最终语义为 B，状态转已处理 |
| 手动修改 | 打开编辑弹窗或行内编辑 |
| 忽略 | 状态转已忽略 |
| 批量确认 | 对已选字段批量处理 |
| 筛选冲突 | 表格只显示冲突字段 |

## 12. 禁止

- 不要多色高饱和设计。
