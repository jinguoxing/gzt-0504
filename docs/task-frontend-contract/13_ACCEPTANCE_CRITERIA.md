# 13｜Acceptance Criteria

## 1. 全局外壳

- [ ] 所有页面左上角只显示 `Semovix`。
- [ ] 副标题为 `语义治理与智能工作台`。
- [ ] 不出现 `Semovix / Xino` 联合 Logo。
- [ ] Xino 不出现在左上角品牌区。
- [ ] 左侧导航顺序固定：首页、AI 工作台、任务、文件、历史记录、与我共享。
- [ ] 顶部栏包含项目选择器、搜索框、通知、帮助、设置、用户头像。
- [ ] 当前页面左侧高亮符合路由规则。

---

## 2. 视觉风格

- [ ] 页面整体为白色 / 浅灰背景。
- [ ] 蓝色作为主品牌色。
- [ ] 状态色只包含蓝、绿、橙、红、灰。
- [ ] 不使用大面积多彩卡片。
- [ ] 中文字体清晰，无伪中文、乱码、错别字。
- [ ] 表格行、按钮、Badge、图标风格统一。

---

## 3. AI 工作台首页

- [ ] 页面默认不显示右侧栏。
- [ ] 中央大标题为“今天想让 Xino 帮你完成什么？”。
- [ ] 输入框是主视觉。
- [ ] 主按钮为“发送给 Xino”。
- [ ] 不出现“生成任务草稿”按钮文案。
- [ ] 常用任务、最近继续、常用资源默认收起。

---

## 4. 草稿抽屉态

- [ ] 不跳转独立草稿页。
- [ ] 中间为任务流式对话，不是聊天气泡。
- [ ] 右侧显示“任务草稿 · 待确认”。
- [ ] 明确显示“尚未开始执行”。
- [ ] 对话修改后，草稿对应项高亮。
- [ ] 底部包含保存草稿、高级配置、创建并开始执行。

---

## 5. 任务中心

- [ ] 默认范围为“我的任务”。
- [ ] 显示范围切换：我的任务、团队任务、全部任务。
- [ ] KPI 包含全部任务、执行中、待我处理、已完成、异常任务。
- [ ] 显示运行中的任务表格。
- [ ] 显示待我处理行动队列，内部包含全部、待确认、待审核、异常、继续执行。
- [ ] 显示最近访问。
- [ ] 显示右侧我的看板、最近活动、最近交付物。
- [ ] 快捷开始视觉权重低于运行中任务和待我处理。

---

## 6. 全部任务列表

- [ ] 页面标题为“全部任务”。
- [ ] 面包屑为“任务 / 全部任务”。
- [ ] 范围切换中“全部任务”高亮。
- [ ] 有视图切换：默认视图、我关注的、异常任务、已归档。
- [ ] 有 KPI 概览：全部任务 128、执行中 36、待确认 17、已完成 75、异常任务 4。
- [ ] 有搜索与筛选工具栏。
- [ ] 表格包含选择框、任务名称、类型、所属项目、当前阶段、整体进度、负责人、优先级、待处理数、状态、最近更新时间、操作。
- [ ] 选中任务后显示批量操作条。
- [ ] 有分页，显示共 128 条和 20 条/页。
- [ ] 右侧栏包含筛选摘要、快捷视图、说明/提示。

---

## 7. 任务执行页

- [ ] 标题为“供应链语义治理闭环任务”。
- [ ] 显示状态“执行中”。
- [ ] 当前任务焦点卡突出阶段“字段语义理解”。
- [ ] 只展开当前阶段结果，历史阶段折叠。
- [ ] 阶段结果卡包括结论摘要、指标组、结果表格、推荐动作。
- [ ] 右侧任务计划为完整阶段进度锚点。
- [ ] 右侧任务计划和任务详情分 Tab。

---

## 8. Review Detail

- [ ] 顶部显示待确认字段、冲突字段、异常字段、已处理。
- [ ] 有 Tab：全部、待确认、冲突、异常、已处理。
- [ ] 主表格展示字段、推断语义、置信度、风险和操作。
- [ ] 右侧字段详情显示样例值、推断依据、相似字段、相关对象、操作记录。
- [ ] 支持确认 A、确认 B、手动修改、忽略。

---

## 9. 完成态

- [ ] 状态为“已完成”。
- [ ] 整体进度 100%。
- [ ] 显示任务已完成总结卡。
- [ ] 显示阶段完成总结时间线。
- [ ] 显示 6 个交付物及用途说明。
- [ ] 显示结果摘要和下一步推荐。

---

## 10. 性能与可用性

- [ ] 列表加载使用骨架屏。
- [ ] 所有主要操作有 loading / disabled 状态。
- [ ] 筛选状态同步 URL query。
- [ ] 空结果有清晰提示和重置筛选入口。
- [ ] 批量操作有确认提示或权限校验。
