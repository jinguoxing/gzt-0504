# 10｜Interaction Rules

## 1. 全局规则

- 所有页面使用同一 AppShell。
- 页面跳转保持上下文，例如从任务中心进入全部任务时保留当前 scope。
- 点击任务名称进入任务详情。
- 点击阶段名称进入任务详情并定位到该阶段。
- 状态、负责人、类型等字段在列表页支持快速筛选。
- 所有“查看全部”必须带上下文筛选条件。

---

## 2. AI 工作台首页

### 2.1 发送给 Xino

```text
输入任务目标
→ 点击发送给 Xino
→ 校验非空
→ 显示 Xino 解析中
→ 生成任务草稿
→ 右侧草稿抽屉展开
```

### 2.2 @ 资源

用户可通过 `@` 选择：

- 数据源
- 文件
- 对象
- 知识库

资源选中后插入输入框并用于草稿解析。

---

## 3. 草稿抽屉

### 3.1 对话调整草稿

```text
用户追加要求
→ Xino 解析变更
→ 中间显示变更摘要
→ 右侧草稿对应项高亮
```

### 3.2 创建并开始执行

```text
点击创建并开始执行
→ loading
→ 创建成功
→ 跳转 /tasks/:taskId
```

失败时显示错误提示和重试。

---

## 4. 任务执行页

### 4.1 当前阶段优先

- 页面只展开当前阶段结果。
- 历史阶段折叠或压缩展示。
- 右侧任务计划作为唯一完整计划锚点。

### 4.2 结果表格操作

| 操作 | 行为 |
|---|---|
| 确认 | 更新字段状态为已确认 |
| 修改 | 打开编辑弹窗或行内编辑 |
| 忽略 | 更新为已忽略 |
| 查看全部字段 | 跳转或打开字段 Review 列表 |

---

## 5. 任务中心

### 5.1 范围切换

```text
我的任务 / 团队任务 / 全部任务
```

切换后刷新：

- KPI
- 运行中的任务
- 待我处理
- 我的看板
- 最近活动
- 最近交付物

### 5.2 KPI 点击

| KPI | 行为 |
|---|---|
| 全部任务 | `/tasks/all?scope=current` |
| 执行中 | `/tasks/all?scope=current&status=RUNNING` |
| 待我处理 | `/tasks/reviews?assignee=me&status=PENDING` |
| 已完成 | `/tasks/all?scope=current&status=COMPLETED` |
| 异常任务 | `/tasks/all?scope=current&status=BLOCKED` |

### 5.3 查看全部

| 模块 | 跳转 | 自动筛选 |
|---|---|---|
| 运行中的任务 | `/tasks/all` | status=RUNNING |
| 待我处理 | `/tasks/reviews` | assignee=me |
| 最近访问 | 最近访问页或弹层 | user=current |
| 最近活动 | 活动日志页 | project=current |
| 最近交付物 | `/resources/deliverables` | sort=recent |

---

## 6. 全部任务列表

### 6.1 筛选

用户可按以下条件筛选：

- scope
- status
- type
- project
- owner
- priority
- time range
- keyword

筛选变更应同步 URL query，方便刷新和分享。

### 6.2 批量选择

当 `selectedTaskIds.length > 0`：

- 显示 BulkActionBar。
- 可执行批量转派、批量归档、批量导出。
- 关闭选择条时清空选择。

### 6.3 列设置

支持控制列显隐：

- 所属项目
- 当前阶段
- 整体进度
- 负责人
- 优先级
- 待处理数
- 最近更新时间

### 6.4 分页

- 默认 pageSize=20。
- 分页切换保留筛选条件。

---

## 7. Review Detail

### 7.1 表格选择

- 点击行选中当前字段。
- 右侧 FieldDetailPanel 更新。
- 支持多选批量处理。

### 7.2 冲突处理

| 操作 | 结果 |
|---|---|
| 确认 A | 最终语义=A，状态已处理 |
| 确认 B | 最终语义=B，状态已处理 |
| 手动修改 | 打开编辑并保存最终语义 |
| 忽略 | 状态已忽略 |

---

## 8. 完成态

### 8.1 交付物操作

| 操作 | 行为 |
|---|---|
| 预览 | 打开文件预览 |
| 下载 | 下载文件 |
| 复用 | 基于交付物生成新任务草稿 |

### 8.2 下一步推荐

点击推荐进入对应新任务草稿或相关页面。

---

## 9. 空态

| 场景 | 文案 | 操作 |
|---|---|---|
| 无运行任务 | 当前没有运行中的任务 | 新建任务 / 从模板创建 |
| 无待处理事项 | 暂无待处理事项，所有任务都在正常推进 | 查看任务中心 |
| 无交付物 | 暂无交付物，任务完成后会显示在这里 | 新建任务 |
| 筛选无结果 | 没有符合条件的任务 | 重置筛选 |

---

## 10. 加载与错误

- 列表使用骨架屏。
- 操作使用按钮 loading。
- 网络失败提供重试。
- 权限不足显示清晰提示，不暴露敏感信息。
